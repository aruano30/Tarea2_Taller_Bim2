# 🛒 Sistema de Tienda

Proyecto desarrollado como tarea utilizando **Spring Boot** y **MySQL**.

## 📌 Descripción

Aplicación web que permite gestionar productos mediante operaciones básicas (CRUD).

## ⚙️ Tecnologías utilizadas

* Java
* Spring Boot
* MySQL
* Thymeleaf
* HTML / CSS

## 🚀 Funcionalidades

* Crear productos
* Listar productos
* Editar productos
* Eliminar productos

## 🛠️ Ejecución del proyecto

1. Clonar el repositorio
2. Configurar la base de datos en MySQL
3. Ejecutar el proyecto desde Spring Boot
4. Acceder desde el navegador
